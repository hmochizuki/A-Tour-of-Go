# このリポジトリについて
このリポジトリは [A Tour of Go](https://go-tour-jp.appspot.com/welcome/1) の個人的学習に利用したもの。
基本的にはツアーの Exercise の回答の記録用。

## ファイル構成

```
A-Tour-of-Go
└ ${exercise name}
  ├ main.go # エクササイズの個人的回答
  └ README.md # エクササイズの回答の解説や個人的メモ（あったりなかったり）
```
